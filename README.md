A full-stack Smart Voting System with real-time polls, leaderboard, admin panel, and secure authentication.

- **Frontend:** React + Vite, Framer Motion
- **Backend:** Node.js + Express
- **Database:** MongoDB

- 🔐 JWT Authentication
- 📊 Live Poll Results
- 🏆 Leaderboard
- 👨‍💼 Admin Panel
- ⏱️ Poll Timers
